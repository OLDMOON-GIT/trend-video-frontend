## ✍️ 작성 구조

**1. 강렬한 훅 (1줄)**
- 시선을 끄는 짧은 문구
- 이모지 활용

**2. 핵심 특징 (3-4줄)**
- 상품의 주요 장점 3-4개를 간결하게
- ✅ 또는 • 기호로 구분
- 각 줄은 짧게 (10-15자)

**3. 구매 링크 (강조 필수)**
```
🛒 구매하기
{product_link}
```
- 반드시 눈에 띄게 배치
- 앞뒤로 빈 줄 추가

**4. 홈 사이트 (강조 필수)**
```
🏠 더 많은 정보
{home_url}
```
- 구매 링크 바로 다음에 배치
- 명확하게 표시

**5. 해시태그 (1줄)**
- 상품 관련 핵심 해시태그 5-10개
- 가장 중요한 것부터 배치

**6. 파트너스 안내 (1줄)**
```
📢 링크 구매 시 소정의 수수료를 받아 채널 운영에 도움이 됩니다
```

### 레이아웃
```
[훅 문구] 🔥

✅ [특징1]
✅ [특징2]
✅ [특징3]

🛒 구매하기
[구매 링크]

🏠 더 많은 정보
[홈 사이트 링크]

#해시태그1 #해시태그2 #해시태그3 #해시태그4 #해시태그5

📢 링크 구매 시 소정의 수수료를 받아 채널 운영에 도움이 됩니다
```

### 스타일 규칙
- **총 길이**: 10줄 내외 (±2줄 허용)
- **문장**: 짧고 명확하게 (한 줄에 15자 이내 권장)
- **이모지**: 각 섹션 시작에만 사용 (과하지 않게)
- **링크**: 반드시 별도 줄로 분리하여 명확하게
- **가독성**: 빈 줄로 섹션 구분
- **톤**: 친근하고 자연스럽게

## 📝 해시태그 가이드

### 선정 기준
1. **상품 직접 관련** (3-4개): 상품명, 카테고리
2. **사용 상황** (2-3개): 언제/어디서 사용
3. **감성 키워드** (2-3개): #가성비템 #강추템 #득템

### 작성 규칙
- 총 5-10개 (너무 많지 않게)
- # 기호 포함
- 공백 없이 붙여쓰기
- 중요한 순서로 배치
- 너무 일반적인 태그는 지양 (#일상, #데일리 등)

## 💡 작성 팁

### 효과적인 훅 만들기
- 질문형: "이런 고민 해보셨나요?"
- 감탄형: "이거 진짜 대박!"
- 문제 제시: "겨울마다 손 시렸다면?"
- 해결 제시: "이제 해결됐어요!"

### 특징 작성 방법
- 기능이 아닌 혜택 중심
- 나쁨: "5000mAh 배터리"
- 좋음: "하루 종일 충전 걱정 없음"

### 링크 배치
- 구매 링크는 항상 상단 (특징 바로 다음)
- 홈 사이트는 구매 링크 바로 다음
- 두 링크 사이에 빈 줄 없이 바로 연결
- 각 링크 앞에 명확한 라벨 (🛒, 🏠)
