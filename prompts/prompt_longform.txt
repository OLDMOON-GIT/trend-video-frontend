**최우선 규칙:**
- 최종 출력 = JSON 객체 1개만
- 문장/이모지/코드펜스/```json 금지
- 순수 JSON만 (백틱 없이)

**출력 금지 (JSON 외 모든 텍스트):**
- 분석/설명/안내/사과 문구 전면 금지
- "~드리겠습니다", "길이 관계상", "생략", "I will", "summary", "omitted" 등
- 사용자에게 묻거나 확인하는 모든 표현

**형식 검증:**
- `{`로 시작 `}`로 끝남
- scenes 정확히 10개 (scene_00_bomb … scene_09_subscribe)
- 씬 1-8: 정확히 2,000자
- ImageFX 프롬프트: 50-70 단어
- 모든 씬 완전한 문장 마무리 (마침표/물음표/느낌표)
- **대사/내레이션: 작은따옴표(')만 사용, 큰따옴표(") 금지** (이스케이프 방지)
- **절대 금지: 메타 정보 포함 금지** (system-reminder, Note:, <system-reminder>, <, 등 모든 시스템 메시지 절대 금지)

**출력 길이 제한 (CRITICAL):**
- Claude 제한: 20,000자 (안전 마진)
- 씬 1-8: 2,000자씩 = 16,000자
- ImageFX: 50-70 단어 = ~2,000자
- JSON 구조: ~1,200자
- ❌ JSON 닫지 않고 끝남 = 파싱 실패
