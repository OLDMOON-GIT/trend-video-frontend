## 🎯 작성 원칙

### 핵심 요구사항
1. **길이**: 10-15줄로 간결하고 임팩트 있게 (핵심만!)
2. **범용성**: 유튜브, 릴스, 페이스북, 틱톡 모든 플랫폼에 동일하게 사용
3. **가독성**: 짧고 강렬한 문장으로 즉각적인 설득력
4. **임팩트**: 핵심 매력만 콕콕 찍어 구매 욕구를 자극
5. **강조 포인트**:
   - 🛒 구매 링크가 가장 눈에 띄게

## ✍️ 작성 구조

**1. 광고 표시 + 강렬한 훅 (1줄)**
- 반드시 "[광고] "로 시작
- 시선을 끄는 짧은 문구
- 이모지 활용

**2. 핵심 특징 (3-5줄)**
- 상품의 핵심 장점 3-5개만 강렬하고 임팩트 있게
- ✅ 기호로 각 항목 시작
- 각 항목은 한 줄로 작성 (빈 줄 없이 연속으로)
- 각 항목은 마침표(.), 느낌표(!), 물음표(?)로 끝내야 함
- 각 줄은 짧고 강렬하게 핵심만 (15-30자 권장)

**3. 구매 링크 (강조 필수)**
```
🛒 구매하기
{product_link}
```
- 반드시 눈에 띄게 배치
- 앞뒤로 빈 줄 추가

**4. 홈 사이트 (강조 필수)**
```
🏠 더 많은 상품보기
{home_url}
```
- 구매 링크 바로 다음에 배치
- 명확하게 표시

**5. 해시태그 (1줄)**
- 상품 관련 핵심 해시태그 8-10개
- 가장 중요한 것부터 배치
- 검색 최적화를 위해 다양한 키워드 포함

**6. 파트너스 안내 (1줄)**
```
이 포스팅은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다.
```

### 레이아웃

**출력 시 반드시 아래 형식을 따르세요:**

```
[광고] [훅 문구] 🔥

✅ [특징1 - 짧고 강렬하게!]
✅ [특징2 - 핵심만!]
✅ [특징3 - 임팩트!]

🛒 구매하기
[구매 링크]

🏠 더 많은 상품보기
[홈 사이트 링크]

#해시태그1 #해시태그2 #해시태그3 #해시태그4 #해시태그5

이 포스팅은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다.
```

**🚨 중요 규칙:**
1. **✅ 항목들은 빈 줄 없이 연속으로 작성** (각 항목은 한 줄씩)
2. **각 ✅ 항목은 마침표(.), 느낌표(!), 물음표(?)로 끝나야 함**
3. **섹션 구분(✅→🛒, 🛒→🏠, 🏠→#, #→📢)에만 빈 줄 하나**
4. **해시태그는 모두 한 줄에 공백으로 구분하여 작성**

### 스타일 규칙
- **총 길이**: 10-15줄 (핵심만 간결하게)
- **문장**: 짧고 강렬하게 (한 줄에 15-25자 권장)
- **이모지**: 각 섹션 시작에 사용하여 시각적 효과 극대화
- **링크**: 반드시 별도 줄로 분리하여 명확하게
- **가독성**: 빈 줄로 섹션 구분
- **톤**: 짧고 강렬하게, 임팩트 위주
- **임팩트**: 핵심 차별화 포인트만 콕콕 찍어서
- **🚨 출력 형식: 반드시 마크다운 코드 블록(```)으로 감싸서 복사-붙여넣기가 가능하도록 출력**

## 🚫 금지 사항

1. **🚨 "[광고] " 표시 없이 시작하기 금지 - 반드시 첫 줄 맨 앞에 "[광고] " 포함**
2. "최고", "1위", "완벽" 등 절대적 표현 금지
3. 검증 안 된 효과/효능 주장 금지
4. 과도한 마케팅 문구 금지
5. **장황하고 긴 문장 금지 (핵심만 짧게!)**
6. **특징 항목 5개 이상 금지 (3-5개만!)**
7. 플랫폼별로 다른 버전 작성 금지 (하나의 텍스트만)
8. **🚨 ✅ 항목들 사이에 빈 줄 넣기 금지 - 연속으로 작성해야 함**
9. **🚨 ✅ 항목이 마침표/느낌표/물음표 없이 끝나기 금지 - 반드시 문장 부호로 끝내야 함**
10. **🚨 마크다운 코드 블록(```) 없이 출력하기 금지 - 반드시 코드 블록으로 감싸야 함**

## 📝 해시태그 가이드

### 선정 기준
1. **상품 직접 관련** (3-4개): 상품명, 카테고리
2. **사용 상황** (2-3개): 언제/어디서 사용
3. **감성 키워드** (2-3개): #가성비템 #강추템 #득템

### 작성 규칙
- 총 8-10개 (핵심 키워드만)
- # 기호 포함
- 공백 없이 붙여쓰기
- 중요한 순서로 배치
- 너무 일반적인 태그는 지양 (#일상, #데일리 등)
- 구체적이고 검색 가능성 높은 태그 위주

## 💡 작성 팁

### 효과적인 훅 만들기
- 질문형: "이런 고민 해보셨나요?"
- 감탄형: "이거 진짜 대박!"
- 문제 제시: "겨울마다 손 시렸다면?"
- 해결 제시: "이제 해결됐어요!"

### 특징 작성 방법
- 기능이 아닌 혜택 중심으로 짧고 강렬하게
- 나쁨: "5000mAh 대용량 배터리로 하루 종일 충전 걱정 없이 최대 48시간 사용 가능합니다!"
- 좋음: "5000mAh로 48시간 사용!"
- 핵심만 짧게, 수치와 혜택만 간결하게

### 링크 배치
- 구매 링크는 항상 상단 (특징 바로 다음)
- 홈 사이트는 구매 링크 바로 다음
- 두 링크 사이에 빈 줄 없이 바로 연결
- 각 링크 앞에 명확한 라벨 (🛒, 🏠)
