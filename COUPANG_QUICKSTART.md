# 쿠팡 파트너스 빠른 시작 가이드

## 🚀 5분 안에 시작하기

### 1단계: API 키 발급 (3분)

1. **쿠팡 파트너스 가입**
   - https://partners.coupang.com 접속
   - 회원가입 및 승인 대기 (보통 1-2일 소요)

2. **API 키 발급**
   - 로그인 후 **마이페이지** → **API 관리**
   - **Access Key 생성** 클릭
   - **Access Key**와 **Secret Key** 복사 (안전하게 보관!)

3. **파트너스 ID 확인**
   - 대시보드에서 본인의 **파트너스 ID** 확인

---

### 2단계: 웹에서 설정 (2분)

1. **애플리케이션 실행**
   ```bash
   cd trend-video-frontend
   npm install
   npm run dev
   ```

2. **쿠팡 페이지 접속**
   - 브라우저에서 http://localhost:3000/coupang 접속

3. **API 키 입력**
   - Access Key 입력
   - Secret Key 입력
   - Tracking ID (파트너스 ID) 입력
   - **저장** 버튼 클릭

4. **연결 테스트**
   - **연결 테스트** 버튼 클릭
   - "✅ 연결 성공!" 메시지 확인

---

### 3단계: 바로 사용하기

#### 상품 검색

1. 검색창에 키워드 입력 (예: "노트북", "이어폰")
2. **검색** 버튼 클릭
3. 검색 결과에서 **링크 생성** 버튼 클릭

#### 파트너스 링크 생성

- 생성된 링크는 자동으로 저장됩니다
- 우측 "생성된 링크" 섹션에서 확인 가능
- **복사** 버튼으로 클립보드에 복사

#### 통계 확인

- 총 링크 수
- 총 클릭 수
- 예상 수익
- 전환율

## 💡 사용 팁

### 검색 키워드 최적화

- ✅ 구체적인 키워드 사용: "노트북" → "맥북 프로 14인치"
- ✅ 브랜드명 포함: "이어폰" → "에어팟 프로 2세대"
- ✅ 모델명 추가: "청소기" → "다이슨 V15 무선청소기"

### 링크 관리

- 생성된 링크는 `data/coupang-links.json`에 저장됨
- 각 링크마다 클릭 수 추적 가능 (수동 업데이트 필요)
- 정기적으로 통계 확인하여 인기 상품 파악

### 수익 극대화

1. **트렌드 상품 검색**: 인기 있는 상품 위주로 링크 생성
2. **계절 상품**: 시즌별 상품 (여름: 선풍기, 겨울: 난방기)
3. **할인 이벤트**: 쿠팡 로켓와우, 타임딜 상품
4. **카테고리 분산**: 다양한 카테고리에서 링크 생성

## 🔧 문제 해결

### "로그인이 필요합니다"
→ `/auth` 페이지에서 로그인

### "API 키를 먼저 설정하세요"
→ API 설정 섹션에서 키 입력 후 저장

### "연결 실패: API 키가 올바르지 않습니다"
→ 쿠팡 파트너스 대시보드에서 키 재확인

### "검색 결과가 없습니다"
→ 다른 키워드로 재검색
